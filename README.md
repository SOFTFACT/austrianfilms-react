# ecoline-react

React + TypeScript Frontend für ECOline (4D REST API auf Port 8282).

## Setup

```bash
npm install
npm run dev
```

Dev-Server läuft auf <http://localhost:5173>, `/api`-Requests werden auf den
4D-Webserver (`http://localhost:8282`) geproxied.

## Build

```bash
npm run build     # → dist/
```
